# CStringProject

This was a project for my Advanced Programming in C++ class. 
I have a created my own <string> library using just arrays and basic C functions. 
I additionally created a Double Linked List to hold the strings from the string library and manipulate them
The current main file takes a file input, takes 5 words then combines them into one, sorts them by ascii values, then outputs them to consle along with the length, capacity, classes created and current
